## **Utility Class**
```System.Math```: คลาสที่รวบรวมฟังก์ชันทางคณิตศาสตร์ 

- **Static Class**: คลาสนี้ถูกประกาศเป็น static ทำให้เราไม่สามารถสร้าง object (instantiate) ด้วยคีย์เวิร์ด new ได้
- **Stateless**: สมาชิกทั้งหมดในคลาสเป็น static members ทำหน้าที่เป็น Tools ในการประมวลผลข้อมูลที่รับเข้ามาทางพพารามิเตอร์แล้ว return ผลออกไป โดยไม่มีการเก็บ state ภายในตัว class

## **Mutable Class**
```System.Text.StringBuilder```: คลาสที่ใช้สำหรับสร้างและจัดการ String ที่มีการแก้ไขบ่อย ๆ

- **State Modification**: เมื่อใช้ Method เช่น .Append(), .Insert(), หรือ .Remove() ข้อมูล String ที่อยู่ใน object เดิมจะถูกแก้ไข
- **Identity**: ตัวแปรที่อ้างอิงถึง StringBuilder ชุดเดิมจะเห็นข้อมูลที่เปลี่ยนไปทันทีหลังจากมีการเรียก Method แก้ไข

## **Immuttable Class**
```System.String```: คลาสพื้นฐานที่ใช้เก็บข้อมูลตัวอักษรใน .NET

- **Read-Only Content**: เมื่อ object ของ String ถูกสร้างขึ้นแล้ว ข้อมูลภายในจะไม่สามารถแก้ไขได้อีก
- **Creation of New Object**: เมื่อใช้ Method ที่ดูเหมือนการแก้ไข เช่น Replace() หรือ .ToUpper() จะสร้าง String object ตัวใหม่ที่มีค่าใหม่ แล้ว return ตัวใหม่ให้เรา ส่วน objecy ตัวเดิมจะนังคงมีค่าเหมือนเดิม
